
#  trivia 

  this is multi topic question bank providing a fast and fair quiz to play in the subjects of your choice users are able to add questions, specify thier category and set thier difficulty , search for questions already added and access a quiz in a specified category it serves as a project for my nanodegre it demonstrates the use of flask , sqlalchemy and python. please use it well, 
### getting started
  
   ##### prerequisites
   `pip, npm, virtual env and python 3.7`
            
   ##### setup
   install the requirments of the backend by running: `pip install -r requirements.txt `
   
   set the enviroment variables:
   `export FLASK_APP=flaskr`
   ` export FLASK_ENV=development`
    
   use `set` for windows instead of export
   copy the database content using:
   
   `psql trivia < trivia.psql`
   
   
   and then install those of the front end by running using :
   `npm install`
    
    
   to launch the app start the backend server using:
   `flask run`
    and then start the frontend program using:
    `npm start`
    
   this is project is hosted locally with default port of 5000 :  ` http://localhost:5000` 
   
   
   
   
   
  ### error handling
  errors are returned as 
  
  ```python
  {
  "success":False
  "error":error code
  "message":error message
  }
  
  ```
   
   the api can return errors of the following types
   `400  bad request`
   `404  resource not found`
   `500  internal server error`
